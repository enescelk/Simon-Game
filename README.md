# Simon-Game
Simon Game with HTML , CSS , Java Script

Simon is an electronic memory game that was invented by Ralph H. Baer and Howard J. Morrison in the late 1970s. It was first launched by Milton Bradley (now part of Hasbro) in 1978 and quickly became a popular toy during the late 1970s and 1980s. The game's name comes from the simple fact that it uses the first names of its creators: "Simon" for Ralph and "Says" for Morrison.

The Simon game consists of a small electronic device with four colored buttons, typically arranged in a circular pattern. The colors of the buttons are usually red, blue, green, and yellow. The device also has a built-in speaker to provide audio feedback.

The game's objective is straightforward: the device plays a sequence of random colors, lighting up the corresponding buttons in that sequence. The player must then repeat the sequence by pressing the buttons in the same order. The sequence starts with just one color and progressively becomes longer and more challenging as the game goes on. Players need to have a good memory to remember the growing sequence of colors.

The game continues until the player makes a mistake and presses the wrong button or fails to repeat the correct sequence within the time limit. When the player loses, the game may produce a distinctive sound or light pattern to indicate the end of the game.

Simon has become an iconic game and has inspired many variations and adaptations over the years, both in physical and digital formats. It's often used as a fun way to challenge and improve memory, concentration, and pattern recognition skills.
